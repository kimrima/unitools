📂 [1번 대분류] PDF 도구: 구현 가능한 전체 리스트 (25종)
AI에게 **"이 리스트를 바탕으로 tools.json을 업데이트하고 기능을 하나씩 활성화해줘"**라고 하시면 됩니다.

1. 파일 형식 변환 (Conversion) - 검색량 1위
PDF to JPG: PDF의 각 페이지를 고화질 이미지(JPG) 파일로 추출.

JPG to PDF: 여러 장의 사진을 하나의 깔끔한 PDF로 병합.

PDF to PNG: 투명 배경이 포함된 이미지 파일로 변환.

PNG to PDF: PNG 이미지들을 PDF로 변환.

PDF to Text: PDF 내의 텍스트 데이터만 긁어내서 메모장 파일로 저장.

HTML to PDF: 웹페이지 주소나 HTML 코드를 PDF 문서로 렌더링.

2. 문서 구조 편집 (Organization) - 사용 빈도 1위
PDF 합치기 (Merge): 여러 개의 PDF 파일을 순서대로 하나로 통합.

PDF 나누기 (Split): 한 PDF 파일에서 특정 페이지들을 각각의 파일로 분리.

PDF 페이지 삭제: 불필요한 페이지를 선택해서 제거.

PDF 페이지 회전: 가로로 누운 페이지를 세로로, 혹은 그 반대로 90도/180도 회전.

PDF 페이지 재정렬: 드래그 앤 드롭으로 페이지 순서를 내 마음대로 변경.

PDF 특정 페이지 추출: 100페이지 중 5페이지와 10페이지만 골라 새 파일 만들기.

3. 최적화 및 보안 (Optimization & Security) - 필수 기능
PDF 압축 (Compress): 고화질을 유지하며 파일 용량만 획기적으로 줄이기.

PDF 암호 설정: 문서 열람 시 비밀번호를 입력해야만 볼 수 있게 잠금.

PDF 암호 해제 (Unlock): 암호가 걸린 문서에서 영구적으로 보안 해제.

PDF 워터마크 삽입: 문서 배경에 "복사 금지"나 회사 로고 이미지 넣기.

PDF 페이지 번호 매기기: 모든 페이지 하단에 1, 2, 3... 번호 자동 삽입.

4. 전문 편집 및 유틸리티 (Advanced Utility)
PDF 여백 제거 (Crop): 문서 테두리의 흰 여백을 잘라내기.

PDF에 텍스트 쓰기: PDF 문서 위에 새로운 텍스트 상자를 얹기.

PDF 서명 (E-Sign): 마우스나 터치로 그린 내 사명을 문서에 삽입.

PDF 메타데이터 수정: 파일 속성의 제목, 작성자, 주제 정보 변경.

PDF 흑백 변환 (Grayscale): 컬러 문서를 흑백으로 바꿔서 인쇄비 아끼기.

PDF 크기 변경 (Resizer): A4 문서를 A3나 Letter 규격으로 변환.

PDF 레이아웃 변경 (N-up): 한 장의 종이에 PDF 2페이지나 4페이지를 모아서 인쇄용으로 배치.

PDF 복구 (Repair): 깨진 PDF 파일의 구조를 분석해 복원 시도.
