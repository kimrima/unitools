이 카테고리는 파일 업로드뿐만 아니라 '복사+붙여넣기' 기반의 도구가 많아 재방문율이 매우 높습니다. 우리가 설치한 **Tesseract.js (OCR 엔진)**와 **SheetJS (엑셀 처리 엔진)**를 활용하여 구현 가능한 30가지 도구를 정리해 드립니다.

📂 [4번 대분류] 텍스트 & 데이터 도구: 구현 가능한 전체 리스트 (30종)
AI에게 **"텍스트 가공 및 데이터 변환 로직을 활성화하고 아래 리스트를 tools.json에 주입해라"**고 지시하세요.

1. OCR 및 텍스트 추출 (OCR & Extraction)
이미지 글자 추출 (OCR): (킬러 기능) 사진 속 글자를 텍스트로 변환 (한/영/일/중 지원).

PDF 텍스트 추출: PDF 문서 내의 모든 텍스트 내용을 긁어오기.

웹페이지 텍스트 추출: URL 입력 시 해당 페이지의 주요 텍스트만 추출.

2. 텍스트 가공 및 정제 (Text Processing)
글자수 세기: 공백 포함/제외 글자수 및 단어수 실시간 계산.

대소문자 변환: 전체 대문자, 소문자, 단어 첫 글자만 대문자로 변환.

줄 바꿈 제거: 텍스트 내의 불필요한 엔터(Enter)를 한 줄로 합치기.

중복 라인 제거: 목록 데이터에서 겹치는 줄을 하나만 남기고 삭제.

텍스트 정렬: 가나다순, 알파벳순, 글자 길이순 정렬.

찾기 및 바꾸기: 특정 단어를 찾아 일괄 변경.

빈 줄 제거: 텍스트 사이의 공백 라인을 모두 삭제.

접두사/접미사 추가: 각 줄의 앞뒤에 특정 문자를 일괄 추가.

3. 데이터 형식 변환 (Data Conversion)
CSV to JSON: 엑셀 데이터를 개발용 JSON으로 변환.

JSON to CSV: JSON 데이터를 엑셀(CSV) 형식으로 변환.

Excel to PDF: 엑셀 파일을 깔끔한 PDF 문서로 저장.

JSON 포맷터: 지저분한 JSON 코드를 보기 좋게 정렬(Beautify).

XML to JSON: 구형 데이터 형식을 신형으로 변환.

YAML to JSON: 설정 파일 형식 상호 변환.

텍스트를 리스트로: 쉼표나 구분자로 된 텍스트를 목록 형태로 변환.

4. 특수 도구 및 생성기 (Special Tools)
가짜 데이터(Dummy) 생성기: 이름, 이메일, 전화번호 등 테스트용 데이터 대량 생성.

로렘 입숨(Lorem Ipsum): 디자인용 임시 텍스트 생성.

Markdown 미리보기: 마크다운 문법을 실시간 HTML로 확인.

정규표현식(Regex) 테스트: 코딩용 정규식 패턴 검증.

URL 인코딩/디코딩: 특수문자가 포함된 URL 주소를 정상화하거나 변환.

Base64 변환: 텍스트나 이미지를 Base64 코드로 변환.

HTML 엔티티 변환: <, > 등을 HTML 특수 기호로 변환.

Diff Checker: 두 텍스트의 차이점을 시각적으로 비교.

단어 빈도수 분석: 어떤 단어가 가장 많이 쓰였는지 통계 내기.

문장 섞기 (Shuffle): 문장이나 단어의 순서를 무작위로 변경.

이진수(Binary) 변환: 텍스트를 0과 1의 이진수로 변환.

모스 부호 변환: 텍스트와 모스 부호 상호 변환.
