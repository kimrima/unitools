📂 [8번 대분류] Miscellaneous & Fun Tools (20종)

1. 텍스트 아트 및 스타일 (Text & Style)
Fancy Text Generator: 일반 텍스트를 유니코드 기호를 사용하여 특이한 폰트(𝔘𝔫𝔦𝔗𝔬𝔬𝔩𝔰, ⓊⓝⓘⓉⓞⓞⓛⓢ)로 변환.

Upside Down Text: 글자를 180도 뒤집어서 보여주는 재미있는 기능 (ɐʍǝsoɯǝ).

Morse Code Translator: 텍스트와 모스 부호 간의 상호 변환 및 사운드 재생.

Binary to Text: 0과 1로 된 이진수를 텍스트로, 텍스트를 이진수로 변환.

Zalgo Text Generator: 글자 위아래로 기괴한 기호가 붙는 '글자 오염' 효과 생성.

2. 랜덤 및 추첨 (Randomizers)
Random Choice Picker: 여러 옵션 중 하나를 무작위로 골라주는 결정 장애 해결 도구.

Wheel of Fortune: 회전하는 돌림판을 돌려 항목을 추첨 (Canvas 애니메이션 활용).

Dice Roller: 3D 또는 2D 주사위를 굴려 무작위 숫자 생성.

Coin Flipper: 앞면/뒷면 동전 던지기 시뮬레이션.

Random Name Generator: 판타지 게임, 소설, 혹은 아기 이름을 위한 무작위 이름 생성.

3. 인터넷 유틸리티 (Internet Utilities)
My IP Address: 사용자의 현재 공인 IP 주소와 대략적인 위치(도시) 표시.

Internet Speed Test: 간단한 다운로드/업로드 속도 측정 위젯 연동.

User Agent Finder: 현재 사용 중인 브라우저와 OS 정보를 상세히 출력.

Redirect Checker: 단축 URL이 최종적으로 어디로 연결되는지 안전하게 확인.

Screen Resolution Checker: 현재 모니터의 해상도와 뷰포트 크기 확인.

4. 기타 유용한 위젯 (Misc Widgets)
Metronome: 악기 연습을 위한 일정한 박자 비트 재생.

White Noise Generator: 집중력을 높여주는 백색소음(비 소리, 파도 소리 등) 재생.

Morse Code Flasher: 스마트폰 플래시나 화면 깜빡임으로 모스 신호 전송.

Simple Drawing Pad: 마우스나 터치로 간단히 메모하고 이미지로 저장.

Color Contrast Checker: 웹 디자인 시 텍스트와 배경의 가독성(대비) 점수 확인.
