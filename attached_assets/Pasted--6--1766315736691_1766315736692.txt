📂 [6번 대분류] 보안 및 개발자 도구: 구현 가능한 전체 리스트 (20종)
AI에게 **"보안 관련 라이브러리(CryptoJS 등)를 활성화하고 아래 리스트를 tools.json에 주입해라"**고 지시하세요.

1. 암호화 및 해시 (Encryption & Hash)
SHA-256 해시 생성기: 텍스트를 가장 대중적인 보안 해시값으로 변환.

MD5 해시 생성기: 파일 무결성 확인용 해시값 생성.

AES 암호화/복호화: 비밀번호를 설정해 텍스트를 암호화하고 푸는 기능.

Base64 인코딩/디코딩: 바이너리 데이터나 텍스트를 웹 안전 포맷으로 변환.

BCrypt 생성기: 비밀번호 해시 처리를 위한 보안 알고리즘.

2. 코드 최적화 및 정렬 (Code Formatting)
HTML Minifier: HTML 코드의 공백을 제거해 용량 최적화.

CSS Minifier: CSS 파일을 한 줄로 압축.

JS Minifier: 자바스크립트 코드 압축 및 난독화 기초.

SQL Formatter: 복잡한 SQL 쿼리를 읽기 좋게 정렬.

JSON Validator: JSON 코드의 문법 오류를 체크하고 시각화.

3. 생성 및 검증 도구 (Generator & Validator)
보안 비밀번호 생성기: 특수문자, 숫자 조합의 강력한 패스워드 자동 생성.

UUID/GUID 생성기: 소프트웨어 개발용 고유 식별자 생성.

JWT 디코더: JSON Web Token의 내용을 분석하고 확인.

Cron Expression 생성기: 서버 예약 작업(Cron) 스케줄 코드를 시각적으로 작성.

Regex 테스터: 정규 표현식이 내 텍스트에 맞는지 실시간 검증.

4. 네트워크 및 유틸리티 (Network & Utils)
IPv4 to IPv6 변환: IP 주소 체계 변환.

What is my IP?: 사용자의 현재 접속 IP 정보와 지역 정보 출력.

HTTP Header 분석기: 특정 URL의 응답 헤더 정보를 분석.

에포크 타임(Epoch Time) 변환: Unix 시간과 일반 시간을 상호 변환.

색상 코드 변환기 (HEX/RGB/HSL): 디자인과 개발 시 필요한 색상 코드 간 변환.
